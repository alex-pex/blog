Si vous avez déjà essayé de copier-coller sur Blogger du code HTML pour l'exposer au monde entier, vous avez du découvrir comme moi que l'éditeur échouait lamentablement. En vue WYSIWYG il ne convertit pas les caractère HTML.

J'ai une solution simple et efficace : [Postable](http://www.elliotswan.com/postable/) convertit les caractère impies, il ne reste plus qu'à coller le résultat là où vous voulliez l'insérer. Une opération qu'on aimerai voir intégrée de base, Blogger a été racheté par Google, c'est censé être pro comme système non ?
